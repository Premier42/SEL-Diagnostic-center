🔧 TROUBLESHOOTING
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Installation failed?
  → Run ./install.sh again (it's safe to re-run)

Can't connect to database?
  → sudo systemctl restart mariadb
  → ./install.sh (re-run installer)

Port 8000 in use?
  → Kill the process: sudo lsof -i :8000
  → Or use different port: php -S localhost:9000 -t public

npm install fails?
  → npm cache clean --force
  → rm -rf node_modules
  → npm install
